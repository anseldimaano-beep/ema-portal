# API Documentation

## Auth
- POST /api/auth/register/ - Register
- POST /api/auth/login/ - Login
- POST /api/auth/refresh/ - Refresh token
- GET /api/auth/profile/ - Profile

## Portal
- GET /api/portal/announcements/ - Announcements
- GET /api/portal/calendar/ - Calendar
- GET /api/portal/faqs/ - FAQs
- POST /api/portal/contact/ - Contact

## Academics
- GET /api/academics/courses/ - Courses
- GET /api/academics/grades/student/ - Student grades
- GET /api/academics/rooms/ - Rooms
