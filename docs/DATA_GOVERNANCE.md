# Data Governance

## Commitments
1. No sensitive data without EEMG authorization
2. Encryption at rest and in transit
3. RBAC with least-privilege
4. Access logging
5. Student data owned by college
6. Privacy policy before Phase 3

## Classification
- Public: Announcements, calendar, catalog
- Low: Contact info, schedules
- High: Grades, records (Phase 3)
